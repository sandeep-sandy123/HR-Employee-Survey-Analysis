# HR Employee Survey Engagement Analysis

## Overview
Analysed 17,600+ employee survey responses across 21 departments to assess workforce engagement using SQL and MS Excel.

## Tools Used
- SQL (SSMS)
- MS Excel

## Key Analysis
- Department-wise response rate calculation
- Low engagement department identification
- Management layer participation analysis (Director, Supervisor, Manager, Staff)
- Top 5 and Bottom 5 departments by survey activity

## Key Finding
Family Justice Center flagged at 95.8% response rate as critical low-engagement area requiring HR intervention.
